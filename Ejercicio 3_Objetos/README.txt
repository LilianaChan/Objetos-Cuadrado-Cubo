Ejercicio 3. Usando la POO crear una clase llamada Cuadrado la cual tendrá las siguientes habilidades:
calcularArea y calcularPerimetro. Crear una clase llamada Cubo que herede de la clase
Cuadrado y que permita las siguientes habilidades: calcularVolumen y calcularPerimetro.

Instrucciones: para ejecutar el programa debera importar el proyecto a Netbeans, y ejecutar la clase principal objetosgeometricos.java 
le aparecera un dialogo donde se pide ingresar el valor de un lado, porterior a eso se le presentara un menu donde seleccionara la operacion que 
desee realizar.
